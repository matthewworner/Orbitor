## Asset Processing Workflow

### Texture Processing
1. Download highest resolution available
2. Convert to appropriate format (JPG for opaque, PNG for transparency)
3. Resize to target resolution (8K or 4K) using high-quality bicubic interpolation
4. Compress using ASTC format for Metal (use Xcode's texture compiler)
5. Verify seamless wrapping for spherical mapping
6. Test in SceneKit with PBR materials

### Audio Processing
1. Download source files (usually WAV or FLAC)
2. Normalize to -12dB to -6dB peak
3. Apply EQ (high-pass at 30Hz, subtle boost at 100-300Hz)
4. Create seamless loops (crossfade last 2 seconds with first 2 seconds)
5. Export as MP3 320kbps or AAC 256kbps
6. Verify loop points with audio editor
7. Test in AVAudioEngine

### Data Processing
1. Fetch TLE data via API
2. Parse TLE format (two-line elements)
3. Cache locally as JSON
4. Implement daily update mechanism
5. Validate data integrity
6. Test SGP4 propagation accuracy
